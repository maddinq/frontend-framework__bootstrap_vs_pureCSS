# GULP Builder Project
generate two sites to compare there performance and load size.
### Site 1 - Twitter Bootstrap
/index.html
### Site 2 - PureCSS
/index-pure.html

## Install
```sh
npm install
bower install
```
Make sure that bower and npm are installed on your system.
##Usage

#### run watcher
```sh
gulp
```

#### run builder (with minify)
```sh
gulp build
```
